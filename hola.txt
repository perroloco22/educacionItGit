**** Eliminar branch ****
Es importante estar ubicado actualmente desde otro branch para poder eliminar el branch deseado
En el siguiente ejemplo tenemos 3 branch y queremos eliminar el branch PrimerBranch y estamos parados actualmente sobre PrimerBranch

>>>git branch
    * PrimerBranch (parado actualmente aqui)
    - main
    -testBranchSecond

>>>git checkout main
>>>git branch --delete PrimerBranch // git branch --delete nameBranch
